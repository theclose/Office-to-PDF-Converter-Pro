1) BẮT ĐẦU NHANH - 3 PHÚT LÀM RA PDF [QS]
================================================================================

Mục tiêu: Từ file Word/Excel/PowerPoint, tạo ra file PDF trong 3 phút.

BƯỚC 1: Mở ứng dụng
- Nhấp đôi vào biểu tượng "Office to PDF Converter Pro" trên màn hình.
- Kiểm tra phiên bản hiển thị trên tiêu đề là v4.1.7.

BƯỚC 2: Thêm file cần convert
- Cách 1: Kéo thả file từ thư mục vào vùng "Kéo thả file vào đây".
  * LƯU Ý: Hỗ trợ kéo thả nhiều file cùng lúc (drag & drop multi-files).
- Cách 2: Nhấn nút "Chọn file" và chọn file từ máy tính.
- Cách 3: Nhấn nút "Folder" để thêm cả thư mục.

BƯỚC 3: Kiểm tra thông báo và danh sách
- Quan sát bảng Log phía dưới: Hiện thông báo "✅ Đã thêm X file(s) thành công".
- File đã thêm sẽ hiển thị trong danh sách ở giữa màn hình.
- Biểu tượng màu xanh lá = Excel, xanh dương = Word, cam = PowerPoint.

BƯỚC 4: Nhấn nút CONVERT
- Nhấn nút "CONVERT" màu xanh ở cuối màn hình.
- Chờ thanh tiến trình chạy đến 100%.

BƯỚC 5: Mở file PDF
- Khi hoàn thành, bạn sẽ thấy thông báo "Hoàn thành".
- File PDF được lưu cùng thư mục với file gốc.
- Nhấn "Mở thư mục" để xem kết quả.

KẾT QUẢ MONG ĐỢI:
- File PDF được tạo ra cùng tên với file gốc, đuôi .pdf
- Nội dung giữ nguyên định dạng, font chữ, hình ảnh.

2) WORD TO PDF [WD]
================================================================================

HƯỚNG DẪN CHUYỂN ĐỔI:

BƯỚC 1: Kéo thả file .doc hoặc .docx vào ứng dụng.
BƯỚC 2: Kiểm tra file đã hiện trong danh sách (biểu tượng xanh dương).
BƯỚC 3: Nhấn nút "CONVERT".
BƯỚC 4: Chờ cho đến khi thanh tiến độ hiện 100%.
BƯỚC 5: Kiểm tra file PDF trong cùng thư mục với file Word gốc.

KẾT QUẢ MONG ĐỢI:
- File PDF giữ nguyên:
  + Định dạng văn bản (in đậm, in nghiêng, gạch chân)
  + Bảng biểu, hình ảnh
  + Header/Footer (đầu trang/cuối trang)
  + Số trang

TIPS:
- Đóng file Word trước khi convert để tránh lỗi "file đang mở".
- Nếu file có nhiều hình ảnh, thời gian convert sẽ lâu hơn.
- File .docx (mới) convert nhanh hơn file .doc (cũ).

LỖI THƯỜNG GẶP:
- Lỗi [WD-01]: Mất font tiếng Việt -> Xem mục [TS-03]
- Lỗi [WD-02]: File đang mở bởi chương trình khác -> Đóng file và thử lại

3) EXCEL TO PDF [XL]
================================================================================

LƯU Ý QUAN TRỌNG:
Excel khác với Word vì dữ liệu có thể tràn ra ngoài trang giấy. Đọc kỹ để
tránh mất dữ liệu sau khi convert.

HƯỚNG DẪN CHUYỂN ĐỔI:

BƯỚC 1: Kéo thả file .xls hoặc .xlsx vào ứng dụng.
BƯỚC 2: Kiểm tra file đã hiện trong danh sách (biểu tượng màu xanh lá).
BƯỚC 3: Chọn chế độ chất lượng:
        - "Cao" (High): Giữ nguyên tất cả, file lớn hơn
        - "Thấp" (Low): Nén nhẹ, file nhỏ hơn
BƯỚC 4: Nhấn nút "CONVERT".
BƯỚC 5: Mở file PDF và kiểm tra tất cả các cột, dòng hiển thị đầy đủ.

KẾT QUẢ MONG ĐỢI:
- Tất cả các sheet được convert (theo mặc định).
- Nội dung nằm gọn trong trang A4.

TIPS TRÁNH BỊ CẮT CỘT:
- Trước khi convert, mở file Excel và vào:
  File > Print > Page Setup > Fit to 1 page wide
- Hoặc thu nhỏ cỡ chữ trong Excel trước khi convert.
- Chọn hướng ngang (Landscape) cho bảng nhiều cột.

WARNING:
Nếu bảng Excel có quá nhiều cột (hơn 20 cột), chữ có thể bị nhỏ đến mức
không đọc được. Hãy thử chia nhỏ bảng hoặc xoay ngang trang.

LỖI THƯỜNG GẶP:
- Lỗi [XL-01]: Bị cắt cột -> Xem mục [TS-04]
- Lỗi [XL-02]: Chỉ thấy 1 sheet -> Ứng dụng convert tất cả sheet theo mặc định
- Lỗi [XL-03]: Bảng trống -> Kiểm tra sheet có dữ liệu không

5) SCAN MODE - CHẾ ĐỘ HÓA ẢNH BẢO MẬT [SM]
================================================================================

SCAN MODE LÀ GÌ?
Scan Mode biến file PDF thành dạng "ảnh chụp", giống như bạn scan (quét)
tài liệu giấy. Toàn bộ nội dung được "làm phẳng" thành hình ảnh.

KHI NÀO NÊN DÙNG?
- Khi bạn muốn GIẢM KHẢ NĂNG người khác copy/edit nội dung.
- Khi cần gửi tài liệu mà không muốn bị chỉnh sửa.
- Khi cần tạo bản PDF giống như scan từ máy photo.

CÁCH BẬT SCAN MODE:

BƯỚC 1: Thêm file cần convert vào ứng dụng.
BƯỚC 2: Tìm và BẬT tùy chọn "Scan Mode" ở phần tùy chọn.
BƯỚC 3: Nhấn nút "CONVERT".
BƯỚC 4: Chờ quá trình hoàn thành.

KẾT QUẢ MONG ĐỢI:
- File PDF trông giống như ảnh chụp.
- Không thể bôi đen chọn chữ để copy.
- Không thể dùng công cụ PDF Editor để sửa.

---------------------------------------------------------------------------
CHỌN DPI (ĐỘ PHÂN GIẢI):
---------------------------------------------------------------------------

DPI là gì?
DPI = "Dots Per Inch" - số điểm ảnh trên mỗi inch.
DPI càng cao -> hình càng nét -> file càng nặng.

Hướng dẫn chọn:

| DPI  | Chất lượng  | Dung lượng | Dùng khi                    |
|------|-------------|------------|-----------------------------|
| 100  | Thấp        | Nhỏ nhất   | Đọc nhanh, không cần nét    |
| 150  | Vừa (*)     | Vừa        | Phù hợp hầu hết trường hợp  |
| 200  | Cao         | Lớn        | Cần đọc chữ nhỏ             |
| 300  | Rất cao     | Rất lớn    | In ấn chuyên nghiệp         |

(*) Mặc định: 150 DPI - cân bằng giữa chất lượng và dung lượng.

---------------------------------------------------------------------------
TÙY CHỌN NÂNG CAO: HIỆU ỨNG SCAN THẬT
---------------------------------------------------------------------------
Nếu bật tùy chọn "Hiệu ứng Scan thật (Grayscale + Noise)", file sẽ:
- Chuyển thành trắng đen (grayscale)
- Thêm nhiễu hạt như giấy thực
- Xoay nghiêng nhẹ giống scan máy photo
- Khó bị phân tích bởi phần mềm AI/OCR hơn

WARNING - HÀM Ý VỀ BẢO MẬT:
Scan Mode KHÔNG ĐẢM BẢO 100% chống copy/edit. Người có kỹ thuật cao
vẫn có thể dùng phần mềm OCR để đọc lại nội dung. Tuy nhiên, nó làm
TĂNG ĐỘ KHÓ KHĂN đáng kể so với PDF thường.

TRADE-OFF DUNG LƯỢNG:
| File gốc       | DPI 150      | DPI 300      |
|----------------|--------------|--------------|
| 500 KB (text)  | 2-5 MB       | 8-20 MB      |
| 2 MB (hình)    | 5-10 MB      | 15-40 MB     |

=> File sau Scan Mode SẼ LỚN HƠN file gốc. Đây là bình thường.

6) PDF TOOLS [PT]
================================================================================

Mở PDF Tools: Nhấn nút "PDF Tools" trên thanh công cụ.

---------------------------------------------------------------------------
6.1) GỘP PDF (MERGE) [PT-01]
---------------------------------------------------------------------------
DÙNG KHI NÀO: Gom nhiều file PDF riêng lẻ thành 1 file duy nhất.

CÁC BƯỚC:
1) Mở PDF Tools > Tab "Chỉnh sửa" > Chọn "Gộp PDF".
2) Thêm các file PDF cần gộp (kéo thả hoặc chọn file).
3) Sắp xếp thứ tự file theo mong muốn.
4) Nhấn "THỰC HIỆN".
5) Chọn nơi lưu file mới.

KẾT QUẢ: 1 file PDF chứa tất cả nội dung các file theo thứ tự đã chọn.

TIP: Kéo thả để đổi thứ tự file trước khi gộp.
NOTE: File kết quả có tên dạng "merged_[tên file đầu].pdf".

---------------------------------------------------------------------------
6.2) TÁCH PDF (SPLIT) [PT-02]
---------------------------------------------------------------------------
DÙNG KHI NÀO: Chia 1 file PDF lớn thành nhiều file nhỏ.

CÁC BƯỚC:
1) Mở PDF Tools > Tab "Chỉnh sửa" > Chọn "Tách PDF".
2) Thêm file PDF cần tách.
3) Chọn cách tách:
   - Tách mỗi trang thành 1 file
   - Tách theo khoảng trang (ví dụ: 1-5, 6-10)
4) Nhấn "THỰC HIỆN".

KẾT QUẢ: Nhiều file PDF nhỏ, được đánh số từ 001, 002...

TIP: Dùng cách tách theo khoảng trang để chia chương sách.
WARNING: Kiểm tra kỹ số trang trước khi tách để tránh mất trang.

---------------------------------------------------------------------------
6.3) TRÍCH XUẤT TRANG (EXTRACT) [PT-03]
---------------------------------------------------------------------------
DÙNG KHI NÀO: Lấy một số trang cụ thể từ file PDF.

CÁC BƯỚC:
1) Mở PDF Tools > Tab "Chỉnh sửa" > Chọn "Trích xuất".
2) Thêm file PDF nguồn.
3) Nhập các trang cần lấy. Ví dụ:
   - "1,3,5" = lấy trang 1, 3, 5
   - "1-5" = lấy trang 1 đến 5
   - "1,3-5,8" = lấy trang 1, trang 3 đến 5, và trang 8
4) Nhấn "THỰC HIỆN".

KẾT QUẢ: File PDF mới chỉ chứa các trang đã chọn.

TIP: Dùng để tạo bản tóm tắt chỉ với các trang quan trọng.
NOTE: File gốc KHÔNG BỊ THAY ĐỔI. Tool tạo file mới.

---------------------------------------------------------------------------
6.4) XÓA TRANG (DELETE) [PT-04]
---------------------------------------------------------------------------
DÙNG KHI NÀO: Loại bỏ một số trang không cần thiết.

CÁC BƯỚC:
1) Mở PDF Tools > Tab "Chỉnh sửa" > Chọn "Xóa trang".
2) Thêm file PDF.
3) Nhập số trang cần xóa. Ví dụ: "2,4,6" hoặc "10-15".
4) Nhấn "THỰC HIỆN".

KẾT QUẢ: File PDF mới KHÔNG có các trang đã xóa.

WARNING: Kiểm tra kỹ trước khi xóa. Nên giữ bản gốc để phòng mất dữ liệu.
TIP: Xóa các trang trống hoặc trang quảng cáo không cần thiết.

---------------------------------------------------------------------------
6.5) XOAY TRANG (ROTATE) [PT-05]
---------------------------------------------------------------------------
DÙNG KHI NÀO: Sửa trang bị ngược, bị xoay ngang sai hướng.

CÁC BƯỚC:
1) Mở PDF Tools > Tab "Chỉnh sửa" > Chọn "Xoay".
2) Thêm file PDF.
3) Chọn góc xoay: 90 độ (phải), 180 độ (ngược), 270 độ (trái).
4) Chọn trang cần xoay hoặc "Tất cả".
5) Nhấn "THỰC HIỆN".

KẾT QUẢ: Các trang được xoay theo góc đã chọn.

TIP: Thường dùng 180 độ để sửa trang bị ngược đầu.
NOTE: Xoay 90 độ để chuyển trang ngang thành dọc hoặc ngược lại.

---------------------------------------------------------------------------
6.6) ĐẢO NGƯỢC THỨ TỰ (REVERSE) [PT-06]
---------------------------------------------------------------------------
DÙNG KHI NÀO: Đảo thứ tự trang (trang cuối thành trang đầu).

CÁC BƯỚC:
1) Mở PDF Tools > Tab "Chỉnh sửa" > Chọn "Đảo ngược".
2) Thêm file PDF.
3) Nhấn "THỰC HIỆN".

KẾT QUẢ: Trang 1 thành trang cuối, trang cuối thành trang 1.

TIP: Hữu ích khi scan tài liệu bị ngược thứ tự.

---------------------------------------------------------------------------
6.7) PDF SANG ẢNH (PDF TO IMAGE) [PT-07]
---------------------------------------------------------------------------
DÙNG KHI NÀO: Xuất mỗi trang PDF thành file ảnh riêng.

CÁC BƯỚC:
1) Mở PDF Tools > Tab "Chuyển đổi" > Chọn "PDF sang Ảnh".
2) Thêm file PDF.
3) Chọn định dạng ảnh: PNG (nét, file lớn) hoặc JPEG (nhỏ hơn).
4) Chọn DPI (150 hoặc 300).
5) Nhấn "THỰC HIỆN".

KẾT QUẢ: Mỗi trang thành 1 file ảnh (page_001.png, page_002.png...).

TIP: Chọn PNG cho tài liệu cần chất lượng cao, JPEG cho ảnh thường.
WARNING: File PDF 100 trang sẽ tạo ra 100 file ảnh.

---------------------------------------------------------------------------
6.8) ẢNH SANG PDF (IMAGE TO PDF) [PT-08]
---------------------------------------------------------------------------
DÙNG KHI NÀO: Gom nhiều ảnh thành 1 file PDF.

CÁC BƯỚC:
1) Mở PDF Tools > Tab "Chuyển đổi" > Chọn "Ảnh sang PDF".
2) Thêm các file ảnh (JPG, PNG, BMP...).
3) Sắp xếp thứ tự ảnh.
4) Nhấn "THỰC HIỆN".

KẾT QUẢ: 1 file PDF, mỗi ảnh là 1 trang.

TIP: Dùng để tạo album ảnh dạng PDF hoặc gom ảnh scan.
NOTE: Nên dùng ảnh cùng kích thước để PDF trông đẹp.

---------------------------------------------------------------------------
6.9) OCR - NHẬN DẠNG CHỮ [PT-09]
---------------------------------------------------------------------------
DÙNG KHI NÀO: Biến PDF scan (dạng ảnh) thành PDF có thể tìm kiếm chữ.

CÁC BƯỚC:
1) Mở PDF Tools > Tab "Chuyển đổi" > Chọn "OCR".
2) Thêm file PDF scan.
3) Nhấn "THỰC HIỆN".
4) Chờ quá trình (có thể mất vài phút).

KẾT QUẢ: File PDF có thể dùng Ctrl+F tìm kiếm nội dung.

WARNING: Độ chính xác OCR phụ thuộc chất lượng ảnh gốc.
- Ảnh mờ, nghiêng, chữ nhỏ -> sai nhiều.
- Ảnh nét, thẳng, chữ rõ -> đúng hơn.

TIP: Scan gốc ở DPI 300 để OCR chính xác hơn.
NOTE: OCR tiếng Việt có thể sai dấu (ă/â, ô/ơ, ư/u).

---------------------------------------------------------------------------
6.10) NÉN PDF (COMPRESS) [PT-10]
---------------------------------------------------------------------------
DÙNG KHI NÀO: Giảm dung lượng file PDF để gửi email hoặc upload.

CÁC BƯỚC:
1) Mở PDF Tools > Tab "Tối ưu" > Chọn "Nén".
2) Thêm file PDF.
3) Chọn mức nén:
   - Thấp: Giảm ít, giữ chất lượng
   - Trung bình: Cân bằng
   - Cao: Giảm nhiều, mất nét
4) Nhấn "THỰC HIỆN".

KẾT QUẢ: File nhỏ hơn. Ví dụ: 10MB -> 2-5MB.

TIP: Chọn mức "Trung bình" cho hầu hết trường hợp.
WARNING: Nén quá mạnh có thể làm ảnh bị vỡ, chữ bị mờ.

---------------------------------------------------------------------------
6.11) ĐẶT MẬT KHẨU (PASSWORD) [PT-11]
---------------------------------------------------------------------------
DÙNG KHI NÀO: Bảo vệ file PDF, chỉ mở được khi nhập mật khẩu.

CÁC BƯỚC:
1) Mở PDF Tools > Tab "Tối ưu" > Chọn "Mật khẩu".
2) Thêm file PDF.
3) Nhập mật khẩu mới (nhớ lưu lại mật khẩu này!).
4) Nhấn "THỰC HIỆN".

KẾT QUẢ: File PDF yêu cầu nhập mật khẩu khi mở.

WARNING - RẤT QUAN TRỌNG:
Nếu QUÊN mật khẩu, KHÔNG CÓ CÁCH nào mở file. 
Hãy lưu mật khẩu ở nơi an toàn.

TIP: Dùng mật khẩu mạnh: kết hợp chữ, số, ký tự đặc biệt.

---------------------------------------------------------------------------
6.12) THÊM WATERMARK [PT-12]
---------------------------------------------------------------------------
DÙNG KHI NÀO: Thêm dòng chữ "MẪU", "CONFIDENTIAL"... trên mỗi trang.

CÁC BƯỚC:
1) Mở PDF Tools > Tab "Tối ưu" > Chọn "Watermark".
2) Thêm file PDF.
3) Nhập nội dung watermark (ví dụ: "BẢN NHÁP", "CÔNG TY ABC").
4) Nhấn "THỰC HIỆN".

KẾT QUẢ: Mỗi trang có dòng chữ mờ xuyên suốt.

8) XỬ LÝ LỖI - TROUBLESHOOTING [TS]
================================================================================

---------------------------------------------------------------------------
[TS-01] KHÔNG CHỌN ĐƯỢC FILE / QUYỀN TRUY CẬP
---------------------------------------------------------------------------
TRIỆU CHỨNG: Nhấn "Chọn file" nhưng không thấy file, hoặc báo lỗi quyền.

NGUYÊN NHÂN:
- File nằm trong thư mục hệ thống (C:\Windows, C:\Program Files).
- File đang bị khóa bởi phần mềm khác.
- Không có quyền đọc file.

CÁCH XỬ LÝ:
1) Copy file sang thư mục khác (Desktop, Documents).
2) Đóng các chương trình đang mở file đó.
3) Nhấn chuột phải vào file > Properties > Security > kiểm tra quyền Read.

---------------------------------------------------------------------------
[TS-02] CONVERT LÂU / TREO ỨNG DỤNG
---------------------------------------------------------------------------
TRIỆU CHỨNG: Thanh tiến trình đứng im, ứng dụng không phản hồi.

NGUYÊN NHÂN:
- File quá lớn (trên 100MB).
- File chứa nhiều hình ảnh/biểu đồ.
- Máy tính cấu hình yếu.

CÁCH XỬ LÝ:
1) Chờ thêm 5-10 phút. File lớn cần thời gian xử lý.
2) Kiểm tra Task Manager (Ctrl+Shift+Esc) xem Excel/Word có đang chạy không.
3) Nếu quá 15 phút vẫn không xong, thử chia nhỏ file rồi convert từng phần.
4) Khởi động lại ứng dụng và thử lại.

---------------------------------------------------------------------------
[TS-03] FONT TIẾNG VIỆT LỖI (Ô VUÔNG, KÝ TỰ LẠ)
---------------------------------------------------------------------------
TRIỆU CHỨNG: Chữ tiếng Việt hiện thành ô vuông hoặc ký tự rác.

NGUYÊN NHÂN:
- File gốc dùng font chưa cài trên máy.
- Font không hỗ trợ Unicode.

CÁCH XỬ LÝ:
1) Mở file gốc (Word/Excel), chọn tất cả (Ctrl+A).
2) Đổi sang font phổ biến: Arial, Times New Roman, hoặc Tahoma.
3) Lưu file và convert lại.
4) Nếu vẫn lỗi, cài thêm font: download font và copy vào C:\Windows\Fonts.

---------------------------------------------------------------------------
[TS-04] EXCEL BỊ CẮT CỘT / CHỮ QUÁ NHỎ
---------------------------------------------------------------------------
TRIỆU CHỨNG: PDF chỉ hiện một phần bảng, cột bị cắt, chữ tí hon.

NGUYÊN NHÂN: Bảng Excel rộng hơn trang A4.

CÁCH XỬ LÝ:
1) Mở file Excel gốc.
2) Vào File > Print > Page Setup.
3) Chọn "Fit All Columns on One Page" (Vừa tất cả cột vào 1 trang).
4) Hoặc chọn hướng giấy Landscape (ngang).
5) Lưu và convert lại.

TIP: Trước khi convert, nhấn Ctrl+P trong Excel để xem Print Preview.

---------------------------------------------------------------------------
[TS-05] SCAN MODE FILE QUÁ NẶNG
---------------------------------------------------------------------------
TRIỆU CHỨNG: File sau Scan Mode lớn gấp 5-10 lần file gốc.

NGUYÊN NHÂN: Đây là bình thường. Scan Mode biến text thành ảnh.

CÁCH XỬ LÝ:
1) Giảm DPI: chọn 100 hoặc 150 thay vì 300.
2) Sau khi Scan, dùng tool "Nén PDF" để giảm dung lượng.
3) Chấp nhận file lớn nếu cần chất lượng cao.

---------------------------------------------------------------------------
[TS-06] FILE CÓ PASSWORD KHÔNG MỞ ĐƯỢC
---------------------------------------------------------------------------
TRIỆU CHỨNG: Ứng dụng báo "File được bảo vệ bằng mật khẩu".

NGUYÊN NHÂN: File gốc đang bị khóa.

CÁCH XỬ LÝ:
1) Mở file bằng Word/Excel/PowerPoint gốc.
2) Nhập mật khẩu.
3) Lưu lại file KHÔNG có mật khẩu (Save As > Tools > General Options > xóa pass).
4) Convert file mới.

NOTE: Ứng dụng KHÔNG phá được mật khẩu. Cần biết mật khẩu gốc.

---------------------------------------------------------------------------
[TS-07] KHÔNG THẤY FILE OUTPUT / KHÔNG LƯU ĐƯỢC
---------------------------------------------------------------------------
TRIỆU CHỨNG: Convert hoàn thành nhưng không tìm thấy file PDF.

NGUYÊN NHÂN:
- Ổ cứng hết dung lượng.
- Thư mục không có quyền ghi.
- Antivirus chặn tạo file.

CÁCH XỬ LÝ:
1) Kiểm tra dung lượng ổ cứng (trên 1GB trống).
2) Thử lưu sang thư mục khác (Desktop hoặc Documents).
3) Tạm tắt antivirus và thử lại.
4) Kiểm tra file ở cùng thư mục với file gốc.

[TS-08] CHIA SẺ LỖI (KHÔNG GỬI ĐƯỢC FILE)

10) TỪ ĐIỂN THUẬT NGỮ - GLOSSARY [GL]
================================================================================

PDF (Portable Document Format):
    Định dạng tài liệu di động. Hiển thị giống nhau trên mọi thiết bị.

DPI (Dots Per Inch):
    Độ phân giải. Số cao = ảnh nét hơn nhưng file lớn hơn.

OCR (Optical Character Recognition):
    Công nghệ nhận dạng chữ từ ảnh. Biến ảnh thành text có thể tìm kiếm.

Scan Mode:
    Chế độ biến PDF thành dạng ảnh để hạn chế copy/edit nội dung.

Rasterize (Hóa ảnh):
    Quá trình biến văn bản/đồ họa thành điểm ảnh (pixel).

Watermark:
    Hình hoặc chữ mờ, hiện trên tất cả các trang để đánh dấu sở hữu.

Merge (Gộp):
    Kết hợp nhiều file thành 1 file duy nhất.

Split (Tách):
    Chia 1 file thành nhiều file nhỏ.

Compress (Nén):
    Giảm dung lượng file bằng cách tối ưu hóa dữ liệu bên trong.

WAL (Write-Ahead Logging):
    Kỹ thuật cơ sở dữ liệu giúp ghi/đọc dữ liệu an toàn và nhanh.

Grayscale:
    Chế độ màu trắng đen (không có màu sắc).

JPEG/JPG:
    Định dạng ảnh phổ biến, nén mất dữ liệu, file nhỏ.

PNG:
    Định dạng ảnh chất lượng cao, không mất dữ liệu, file lớn hơn.

Flatten (Làm phẳng):
    Gộp tất cả các lớp (layer) thành 1 lớp duy nhất.
